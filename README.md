# FLAT

## Overview
Ruby on Railsで作成したポートフォリオ用サイト。訪れた場所を写真と動画付きで投稿し、ユーザー間で共有できるサービス。

## 使用言語／パッケージ
* Ruby version 2.5.5
* Rails version 5.2.3
* Nodejs version 12.4.0
* Yarn version 1.19.1
* Webpacker
* Vue.js

## Vue.js 使用ライブラリ
* axios
* vue-router
* vuex
